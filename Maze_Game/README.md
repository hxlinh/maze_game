# Maze Game

## Overview
![Maze game interface](https://github.com/user-attachments/assets/94bf4502-c660-40d9-8f77-ef9c7985c94e)
A simple browser-based maze game where the player navigates a randomly generated maze to reach the goal. The game features keyboard and swipe controls, hints, and customizable difficulty.

## Features
- Random maze generation with increasing difficulty
- Player movement using arrow keys (or WASD) and swipe gestures (mobile)
- Visual hints to help solve the maze
- Customizable sprites for player, goal, and hints
- Responsive canvas rendering

## How to Play
1. Select the desired difficulty from the dropdown.
2. Use arrow keys (or WASD) to move the player sprite through the maze.
3. Click the hint button to reveal part of the solution path.
